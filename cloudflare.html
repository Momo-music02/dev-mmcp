Comment Cloudflare est utilisé

Cloudflare protège le site MMCP contre les attaques DDoS, optimise la performance via le CDN, et gère le DNS du domaine. Il permet aussi d'utiliser des Workers pour des fonctions avancées (ex: redirections, cache).

Étapes d'intégration :

  1. Ajout du domaine sur Cloudflare.
  2. Configuration du DNS pour pointer vers Firebase Hosting.
  3. Activation des règles de sécurité et du CDN.
  4. Utilisation de Workers pour des besoins spécifiques.
