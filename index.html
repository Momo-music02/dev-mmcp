Bienvenue sur l’espace développeur MMCP

Ce site regroupe toutes les informations pour comprendre, modifier et déployer le distributeur musical Morgann Music CP.
Découvre comment chaque technologie s’intègre dans le projet et accède à la documentation détaillée.

Firebase

Firebase gère l’authentification, la base de données (Firestore), le stockage des fichiers et l’hébergement du site. Il permet une gestion sécurisée et scalable des utilisateurs, des releases et des médias.

Cloudflare

Cloudflare protège le site contre les attaques, optimise la rapidité grâce à son CDN, gère le DNS et permet d’utiliser des Workers pour des fonctions avancées (redirections, cache, sécurité).

Google Apps Script

Apps Script automatise des tâches (emails, génération de documents, chat AI) et expose des API personnalisées pour MMCP. Les scripts sont connectés au front-end via des requêtes HTTP sécurisées.

Déploiement

Le déploiement du site se fait via Firebase Hosting, avec gestion du code sur GitHub et sécurisation Cloudflare. Toutes les étapes sont documentées pour faciliter la mise à jour et la maintenance.
