Déploiement du site MMCP

Le déploiement se fait principalement via Firebase Hosting. Les modifications sont versionnées sur GitHub, puis déployées après validation. Cloudflare assure la sécurité et la rapidité du site.

Étapes de déploiement :

  1. Développement local dans le dossier dev/.
  2. Commit et push sur le dépôt GitHub dev-mmcp.
  3. Déploiement sur Firebase Hosting avec firebase deploy.
  4. Vérification du site via Cloudflare (CDN, sécurité, DNS).
