Comment Firebase est utilisé

Firebase est utilisé pour l'authentification, la base de données (Firestore), le stockage de fichiers et l'hébergement du site MMCP. Les règles de sécurité sont définies dans firestore.rules et storage.rules.

Étapes d'intégration :

  1. Création d'un projet Firebase.
  2. Ajout du SDK Firebase dans le code front-end.
  3. Configuration de l'authentification (email, Google, etc.).
  4. Utilisation de Firestore pour stocker les données utilisateurs, releases, etc.
  5. Utilisation de Firebase Storage pour les fichiers audio/images.
  6. Déploiement via firebase.json.
