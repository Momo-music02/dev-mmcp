Comment Apps Script est utilisé

Google Apps Script permet d'automatiser des tâches (emails, génération de documents, chat AI) et d'exposer des API personnalisées pour MMCP. Les scripts sont stockés dans scripts/ et peuvent être déployés comme Web Apps.

Étapes d'intégration :

  1. Développement des scripts dans Google Apps Script Editor.
  2. Déploiement en tant que Web App pour exposer des endpoints (ex: temp_genai.gs).
  3. Connexion avec le front-end via des requêtes HTTP.
  4. Gestion des clés API et des permissions dans Apps Script.
